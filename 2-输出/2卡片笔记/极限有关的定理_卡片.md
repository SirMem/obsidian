# 🍎重点摘抄
## 无穷小乘有界函数定理
如果函数f(x) 在某个点 c 处的极限为零，而函数g(x) 在该点附近有界，那么乘积 f(x)⋅g(x) 在 c 处的极限也为零。

## 三明治定理
**夹逼定理（又称挤压定理）：** 如果存在另外两个函数 g(x) 和 h(x)，使得在某个区间内g(x)≤f(x)≤h(x)，并且 lim(x→c​) g(x) =lim(x→c​) h(x)=L，那么就有lim(x→c​) f(x)=L。
